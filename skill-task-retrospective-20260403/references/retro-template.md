# 复盘模板速查

## 快速复盘（5分钟）

```
# 今日复盘 — YYYY-MM-DD

## 做得好的
1. 

## 需要改进
1. 

## 下次这样做
1. 
```

## 标准复盘（15-30分钟）

```markdown
# 任务复盘 — YYYY-MM-DD HH:MM

## 📋 基本信息
- 任务: <描述>
- 类型: <功能/Bug/研究/流程/...>
- 时长: <N小时/分钟>
- 复杂度: ⚡×N

## 🔄 执行回顾
<简要描述执行过程>

## ✅ 成功要素
1. <做得好的>
2. 
3. 

## 🚨 教训
1. <需要改进的>
2. 
3. 

## 📌 规律提炼
<从具体案例抽象成原则>

## 🎯 下次行动计划
- [ ] <具体改进行动>
- [ ] <对应哪个教训>

## 💡 灵感记录
<任务过程中产生的其他想法>
```

```markdown
# 团队复盘 — 项目名 — YYYY-MM-DD

## 参与人
-

## 目标回顾
- 原定目标: 
- 实际结果:

## What went well (WWW)
-

## What could be improved (WBI)
-

## Action items (AI)
| 行动 | 负责人 | 截止日期 |
|------|--------|----------|
|      |        |          |

## Team health check
- 沟通: 😊😐😟
- 协作: 😊😐😟
- 工具: 😊😐😟
```

## 触发时机

| 时机 | 用哪个模板 |
|------|-----------|
| 每日结束时 | 快速复盘 |
| 单次任务完成 | 标准复盘 |
| 项目/里程碑完成 | 多人项目复盘 |
| 踩坑后立刻 | 教训专项复盘（精简版） |
