# AI Code Review

## 概述

帮助用户审查和优化AI生成的代码，解决AI开发过程中的代码质量问题。

## 触发场景

- "帮我审查这段代码"
- "检查AI写的代码有没有问题"
- "review一下这个代码"
- "代码有没有安全漏洞"
- "优化这段AI生成的代码"

## 审查维度

### 1. 正确性检查
- 逻辑错误
- 边界条件处理
- 异常处理完整性

### 2. 安全审查
- SQL注入漏洞
- XSS安全问题
- 敏感信息泄露
- 认证授权问题

### 3. 性能评估
- 时间复杂度分析
- 空间复杂度分析
- 数据库查询优化
- 缓存策略建议

### 4. 代码质量
- 代码可读性
- 命名规范
- 函数/类设计
- 重复代码检测

### 5. 最佳实践
- 依赖项安全性
- 测试覆盖建议
- 文档完整性
- API设计合理性

## 输出格式

```markdown
## 审查报告

### 问题列表
| 严重程度 | 位置 | 问题描述 | 建议修复 |
|---------|------|---------|---------|
| 🔴 高   | xxx  | xxx     | xxx     |

### 优化建议
- ...

### 总结
- ...
```

## 注意事项

- 只提供建议，不直接修改用户代码
- 优先关注高严重程度问题
- 结合具体编程语言的最佳实践
- 考虑代码的实际使用场景
